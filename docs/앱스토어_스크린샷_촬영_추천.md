# 앱 스토어 스크린샷 10장 – 어떤 화면 찍을지 (추천)

앞 3장은 **메인(첫인상)**, 나머지 7장은 **기능·신뢰감**이 보이도록 구성했습니다.  
고급스럽고 효율적으로 보이려면 **지도·공간·채팅**을 앞에, **동네생활·동네부탁·프로필**을 뒤에 두는 구성을 추천합니다.

---

## 앞 3장 (메인 – 가장 먼저 보이는 것)

| 순서 | 찍을 화면 | 이유 |
|------|-----------|------|
| **1** | **홈 (지도)** | 지도 + 마커 + 공간 카드. “주변 보관 공간 한눈에”가 바로 보여서 고급·효율 느낌. |
| **2** | **공간 상세** | 사진·가격·설명이 잘 나온 공간 하나. 품질 있는 리스팅이 보이면 서비스가 믿어짐. |
| **3** | **채팅 (보관 요청/약속)** | 보관요청 카드나 깔끔한 대화가 보이는 채팅 화면. “예약·소통이 쉽다”는 인상. |

---

## 나머지 7장 (기능·다양성·완성도)

| 순서 | 찍을 화면 | 이유 |
|------|-----------|------|
| **4** | **홈 (공간 리스트)** | 지도가 아닌 리스트/카드 뷰. “검색·필터로 골라쓴다”는 효율성. |
| **5** | **동네생활** | 사진 있는 게시글·댓글이 보이는 화면. 커뮤니티가 살아 있음. |
| **6** | **동네부탁** | 부탁 목록 또는 부탁 카드 한두 개. “보관 말고 다른 도움도 있다”는 확장감. |
| **7** | **채팅 목록** | 채팅 탭 – 대화 목록이 정돈된 화면. 실시간 소통이 가능해 보임. |
| **8** | **즐겨찾는 장소** | 프로필 → 즐겨찾는 장소. 자주 쓰는 공간 관리 = 효율적인 이용. |
| **9** | **공간 상세 (다른 공간)** | 2번과 다른 공간 하나 더. 다양한 공간이 있다는 인상. |
| **10** | **프로필** | 닉네임·나의 포스·설정 등. 앱이 완성도 있게 마무리된 느낌. |

---

## 촬영 시 팁 (고급·효율 느낌)

- **데이터 채우기**: 지도에 마커 몇 개, 채팅에 대화 2~3줄, 동네생활에 게시글 1~2개 있으면 훨씬 자연스럽고 완성도 있어 보임.
- **공간 상세**: 사진이 크고, 가격·설명이 읽기 좋은 공간을 골라서 찍기.
- **채팅**: 욕·개인정보 없이, “안녕하세요 / 보관 요청 드려요 / 네 가능해요” 정도의 짧은 대화가 ideal.
- **로그인 상태**: 로그인된 상태에서 찍으면 채팅·즐겨찾기·프로필이 다 보여서 추천.

---

## 요약 (10장 체크리스트)

| # | 화면 | 탭/경로 |
|---|------|---------|
| 1 | 홈 지도 | 홈 탭 (지도 뷰) |
| 2 | 공간 상세 | 공간 하나 탭 → 상세 |
| 3 | 채팅 (보관요청/대화) | 공간 상세 → 채팅하기 |
| 4 | 홈 리스트/카드 | 홈 탭 (리스트 뷰) |
| 5 | 동네생활 | 동네생활 탭 |
| 6 | 동네부탁 | 동네부탁 탭 |
| 7 | 채팅 목록 | 채팅 탭 |
| 8 | 즐겨찾는 장소 | 프로필 → 즐겨찾는 장소 |
| 9 | 공간 상세 (다른 공간) | 다른 공간 상세 |
| 10 | 프로필 | 프로필 탭 |

이 순서로 찍으면 “지도로 찾고 → 공간 보고 → 채팅으로 예약하고 → 동네생활·동네부탁까지 쓸 수 있는 앱”이라는 흐름이 잘 전달됩니다.
