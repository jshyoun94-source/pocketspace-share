# 보관 단계 및 약속잡기 가이드

## 1. 공간 레벨: 대여가능/대여불가

- **대여가능**: 공간 등록자가 모든 사용자에게 보이게 설정 (기본값)
- **대여불가**: 대여 중단, 진행 중인 채팅에도 "대여불가" 표시
- 공간 수정 화면에서 실시간 변경 가능

## 2. 거래 단계 (채팅 단위)

| 단계 | 설명 |
|------|------|
| 보관신청중 | 사용자가 약속잡기로 보관 요청 전송 |
| 약속중 | 공간대여자가 수락하기 클릭 |
| 보관중 | 물건 보관 진행 중 |
| 보관종료 | 반납 완료, 상호 평가 가능 |

## 3. 약속잡기 플로우

1. **사용자**: "약속잡기" 클릭 → 모달에서 물건 사진(선택), 보관 날짜/시간(필수) 입력
2. **보관 요청** 전송 → 채팅에 보관요청 메시지 표시
3. **공간대여자**: 메시지 내 "수락하기" 클릭 → 단계가 "약속중"으로 변경
4. **공간대여자**: 거래 카드에서 "보관 시작" → "보관중"
5. **공간대여자**: "보관 종료" → "보관종료", 양측 평가 모달 표시

## 4. UI 변경사항

- 공간 카드 이미지: 56×56 정사각형
- 포스페이 버튼 제거
- 약속잡기: 사용자만 노출, 보관요청 전송 후 수락하기 버튼으로 응답
