# 이미지 업로드 문제 해결 가이드

## 1. Firebase Console에서 확인

1. **Firebase Console** → 프로젝트 `tangential-sled-352810` 선택
2. **Storage** 메뉴 클릭
3. Storage가 **사용 설정**되어 있는지 확인
4. 버킷 이름 확인 (예: `tangential-sled-352810.appspot.com` 또는 `tangential-sled-352810.firebasestorage.app`)

## 2. .env 설정 확인

`.env` 파일에서 다음 값이 올바른지 확인:

```
EXPO_PUBLIC_FIREBASE_STORAGE_BUCKET=tangential-sled-352810.firebasestorage.app
```

**Firebase Console > 프로젝트 설정 > 일반**에서 `storageBucket` 값을 그대로 복사해서 사용하세요.

## 3. 배포

```bash
firebase deploy --only functions
```
