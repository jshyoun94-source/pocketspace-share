# Guideline 4.8 – 심사 팀 답글 초안 (Apple 연동 없이 재제출 시)

Sign in with Apple을 아직 구현하지 않고, 기존 로그인(네이버·카카오·구글)으로 4.8을 충족한다고 설명할 때 사용하는 **복사용** 초안입니다.

**사용 방법:** App Store Connect에서 아래 경로로 들어가 **"심사 팀에 대한 답글"** (또는 "Reply to App Review") 칸에 아래 본문을 복사해 붙여넣기.

---

## App Store Connect에서 답글 넣는 위치

### 1) 거부된 앱에 대해 답글할 때 (재제출 전)

1. **App Store Connect** (appstoreconnect.apple.com) 로그인
2. **앱** → 해당 앱(PocketSpace) 선택
3. 왼쪽 메뉴에서 **"앱 심사"** 또는 **"App Review"** 클릭
4. **"해결 센터"** / **"Resolution Center"** 또는 **"심사 중" / "Rejected"** 상태인 버전이 보이면 해당 항목 클릭
5. Apple이 보낸 **거부 메시지** 아래에 **"심사 팀에 대한 답글"** / **"Reply to App Review"** 입력 칸이 있음 → 여기에 위 본문 붙여넣기
6. 저장 후, 수정 사항 반영한 **새 빌드**를 올리고 **"심사에 제출"** / **"Submit for Review"** 클릭해 재제출

### 2) 새 버전 제출 시 미리 적어 둘 때

- **"앱 심사"** → **"버전 정보"** 또는 **"Version Information"** 화면에서  
  **"심사 메모"** / **"Notes for App Review"** (또는 "App Review Information") 필드가 있으면  
  그곳에 위 4.8 관련 설명을 붙여넣어도 됩니다.  
  (화면 구성은 Apple 업데이트에 따라 "Notes", "Review notes", "Contact information" 등으로 나뉠 수 있음)

### 3) 찾기 어렵을 때

- **Resolution Center** = 거부/메시지가 오는 곳, 여기서 **Reply** 버튼 또는 답글 입력 칸을 찾으면 됨.  
- **Submit for Review** 하기 직전 단계에서 **"Add Note"**, **"Notes"**, **"심사 팀에 대한 메모"** 같은 항목이 있으면 같은 내용을 거기에도 넣을 수 있음.

---

## 본문 (복사용)

Apple 심사 팀에,

Guideline 4.8(동등한 로그인 옵션) 관련하여, PocketSpace는 현재 네이버·카카오·구글 소셜 로그인을 제공하고 있으며, 이 옵션들이 해당 가이드라인의 요구 사항을 충족한다고 판단합니다.

1. **데이터 수집 최소화**  
   네이버·카카오·구글 로그인은 서비스 운영에 필수적인 최소한의 정보(이름, 이메일 주소)만 수집하며, 과도한 개인 정보를 요구하지 않습니다.

2. **이메일의 타인 비노출**  
   PocketSpace에서는 이메일 주소를 **타인에게 전혀 노출하지 않습니다.** 다른 이용자에게는 닉네임(또는 이름)만 표시되며, 이메일은 본인 프로필 화면에서만 본인이 볼 수 있습니다. 따라서 계정 설정 시 "이메일을 비공개로 할 수 있는 옵션"에 해당하는 동등한 보호가, 앱 설계상 이미 적용되어 있습니다.

3. **광고 목적 수집 없음**  
   PocketSpace는 사용자의 명시적 동의 없이 앱 내 활동을 광고 목적으로 수집하거나 제3자에게 제공하지 않습니다. 데이터는 서비스 제공 및 개선 목적으로만 사용됩니다.

저희 앱은 한국 사용자를 주요 대상으로 하며, 네이버·카카오·구글 로그인이 한국에서 가장 보편적인 인증 수단입니다. 위 로그인 옵션들이 4.8의 취지(사용자 프라이버시 보호)를 충분히 만족한다고 판단하여, 현재는 추가 로그인 옵션 없이 재제출드립니다. Sign in with Apple은 추후 버전에서 검토할 예정입니다.

확인 부탁드립니다.

---

## 주의사항

- **2번 문단:** 앱에는 "이메일 비공개" 토글이 없고, 대신 **이메일을 타인에게 아예 노출하지 않는** 방식으로 동작합니다. 위 초안은 이 사실을 그대로 반영한 내용입니다.
- **영문 제출:** 심사 팀이 영문을 요구할 수 있으므로, 필요 시 위 내용을 영문으로 번역해 동일하게 "심사 팀에 대한 답글"에 넣으면 됩니다.
